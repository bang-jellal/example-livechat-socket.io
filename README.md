# Example Livechat using Laravel + Socket.io

## Requirements

1. [WAMPP/XAMPP](https://www.apachefriends.org/index.html)
2. [Composer](https://getcomposer.org/)
3. [NodeJS + NPM](https://nodejs.org/en/)

## Installation

```bash
$ composer install
$ npm install
$ npm run dev
$ npm run socket-io-server
$ php artisan serve
```
